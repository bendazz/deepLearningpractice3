Question 1 short answer

Consider the following tiny vocabulary, which maps each word to its token ID: 

{
  "the": 0,
  "cat": 1,
  "dog": 2,
  "sat": 3,
  "ran": 4,
  "on":  5,
  "mat": 6
}

Using this vocabulary, encode the sentence

"the cat sat on the mat"

as a sequence of token IDs.

Answer

Encoded sequence: $[\,0,\ 1,\ 3,\ 5,\ 0,\ 6\,]$

Reasoning: replace each word with its ID from the vocabulary, in order:

  • "the" → 0
  • "cat" → 1
  • "sat" → 3
  • "on" → 5
  • "the" → 0
  • "mat" → 6
Question 2 short answer

Using the same vocabulary as Question 1, suppose we define a (toy, not semantically meaningful) embedding that maps each token ID $i$ to a 2-dimensional vector by the formula

$$ e(i) = (\,i + 1,\ 2i - 1\,). $$

What is the embedded vector for the word "sat"?

Answer

$e(\text{"sat"}) = (4,\ 5)$

Reasoning:

  • From the vocabulary, "sat" has token ID $i = 3$.
  • Plug into the formula: $e(3) = (3 + 1,\ 2 \cdot 3 - 1) = (4,\ 5)$.
Question 3 short answer

Using the same vocabulary and embedding formula as before $$ e(i) = (\,i + 1,\ 2i - 1\,), $$ compute the embedding matrix for the sentence

"dog sat on mat"

using the convention that each row of the matrix is the embedding of one token (so the result has shape $T \times d$, where $T$ is the number of tokens and $d$ is the embedding dimension).

Answer

The sentence has 4 tokens and the embedding dimension is 2, so the embedding matrix has shape $4 \times 2$:

$$ X \;=\; \begin{bmatrix} 3 & 3 \\ 4 & 5 \\ 6 & 9 \\ 7 & 11 \end{bmatrix} $$

Reasoning: look up each token's ID, apply the embedding formula, and stack the results as rows:

  • "dog" → id 2 → $e(2) = (3,\ 3)$
  • "sat" → id 3 → $e(3) = (4,\ 5)$
  • "on" → id 5 → $e(5) = (6,\ 9)$
  • "mat" → id 6 → $e(6) = (7,\ 11)$
Question 4 code from memory

Suppose you are given a vocabulary as a Python list of tokens. For example, the vocabulary we've been using in the previous questions corresponds to the list 

vocab = ["the", "cat", "dog", "sat", "ran", "on", "mat"]

Write a Python function that takes such a vocab list as input and returns a dictionary mapping each token to its index in the list.

Answer
def build_vocab_dict(vocab):
    vocab_dictionary = {}
    for i, token in enumerate(vocab):
        vocab_dictionary[token] = i
    return vocab_dictionary

Reasoning:

  • enumerate(vocab) yields (i, token) pairs where i is the position of token in the list.
  • Inside the loop, we add an entry token → i to the dictionary.
  • After the loop, every token in the vocabulary has been mapped to its index, and we return the resulting dictionary.

For example, applied to the vocabulary above, calling build_vocab_dict(vocab) would produce {"the": 0, "cat": 1, "dog": 2, "sat": 3, "ran": 4, "on": 5, "mat": 6}, which matches the token-ID mapping used in the earlier questions.
